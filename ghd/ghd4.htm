How to download and Install GHD Sports APK in Amazon Firestick/Fire TV

GHD Sports also run in big screen with Firestick or Fire TV you can run this app on Android TV Box, Android Smart TV, NVIDIA Shield, Ruko, Android Smart Watch etc.

If you are not aware how to install in Firestick? We have given complete installation procedure just follow steps:

1. First of all go to Home Screen and then click on Settings

2. After that you get new screen click on Device option

3. Click on Developer Options button

4. Click on Apps from Unknown Sources button

5. Press button Turn On

6. Return to Home & click on Search

7. Write Downloader in search box

8. Tap on Downloader

9. Click Open button

10. Click on Allow button

11. Click on OK button

12. Go to Settings in left

13. Click on the box to Enable JavaScript

14. Click on Browser

15. Tap on the bar on top to write

16. Enter exact URL for GHD Sports download (http://bit.ly/2FabSfV)

17. Now wait until the download of APK file is complete

18. Now tap on the Install button

19. GHD Sports APK is installed now, click on the Done button

20. Return to Home and go to Your Apps & Channels and click See All

21. You can see the GHD Sports app at the bottom
